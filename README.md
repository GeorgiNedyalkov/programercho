# Programercho

A game where you use programming concepts as puzzles learning more about programming and the computer.
Written in C from scratch.
The demo of the game should have one puzzle type with several different puzzles.

The game should be fun, funny, interesting but not patronizing and too educational.

The puzzles can be abstract programming concepts.

## Puzzle ideas
- Logic Gates
- Bugfixing

## Gameplay loop

You collect ones and zeros (lightbulbs).

You find a part. You build a thing. You solve a puzzle. You make a gadget

You use the gadget to build a thing. Use it for another puzzle. Get a better gadget.
